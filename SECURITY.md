# Security Policy

## FR - Signaler une vulnerabilite

Pour toute vulnerabilite de securite, contactez-nous en prive:

- Email: [johan.ledoux@rootage.fr](mailto:johan.ledoux@rootage.fr)
- Objet recommande: `[SECURITY] Nom du repository - Resume`

Merci de ne pas ouvrir de ticket public pour une faille non corrigee.

## EN - Reporting a vulnerability

For any security vulnerability, please contact us privately:

- Email: [johan.ledoux@rootage.fr](mailto:johan.ledoux@rootage.fr)
- Suggested subject: `[SECURITY] Repository name - Summary`

Please do not open a public issue for an unpatched vulnerability.

## FR - Delais

Nous accusons reception sous **72 heures** et revenons vers vous avec un plan de remediation.

## EN - SLA

We acknowledge receipt within **72 hours** and follow up with a remediation plan.
